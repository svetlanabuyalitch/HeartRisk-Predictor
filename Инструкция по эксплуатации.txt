Назначение программы
HeartRisk Predictor — это веб-сервис для предсказания риска сердечного приступа на основе медицинских данных пациентов. Программа принимает CSV-файл с данными, обрабатывает его с помощью обученной модели машинного обучения и возвращает предсказания для каждого пациента.

Программа работает как локальный веб-сервер, к которому можно обращаться через браузер или через API-запросы.

В папке проекта находятся следующие файлы:

app.py — главный файл программы
Это сердце вашего приложения. Он:

Запускает веб-сервер

Загружает обученную модель

Обрабатывает загруженные CSV-файлы

Отдаёт веб-страницы пользователю

Возвращает результаты предсказаний

Взаимодействует с HTML-шаблонами

model.py — вспомогательный файл для работы с изображениями
Содержит код для обработки изображений (из учебного примера). Этот файл не используется для предсказаний риска, но необходим для работы программы.

templates/ — папка с веб-страницами
Содержит HTML-файлы, которые видит пользователь в браузере:

predict_form.html — страница с формой для загрузки CSV-файла

result.html — страница с результатами предсказаний

error.html — страница с сообщением об ошибке

best_heart_attack_model.bin — обученная модель
Файл с моделью машинного обучения, которая:

Была обучена на исторических данных

Умеет определять риск сердечного приступа

Загружается при запуске сервера

tmp/ — папка для временных файлов
Сюда сохраняются:

Загруженные пользователем CSV-файлы

Результаты предсказаний в формате JSON

test.py — простой тест для проверки работы сервера
Проверяет, запущен ли сервер и отвечает ли он.

test_client.py — продвинутый тест для проверки API
Отправляет тестовые запросы к серверу и проверяет, правильно ли работают предсказания.

Работа с программой:

Откройте браузер

Перейдите по адресу: http://localhost:8000

Вы увидите страницу с заголовком "Heart Attack Risk Prediction" и формой загрузки файла.

Подготовьте CSV-файл с данными пациентов в правильном формате (см. раздел "Форматы данных").

Нажмите кнопку "Выберите файл" и укажите путь к вашему CSV-файлу.

Нажмите кнопку "Загрузить и предсказать".

Дождитесь обработки — на экране появится страница с результатами:

Общее количество обработанных записей

Сколько пациентов с низким и высоким риском

Таблица с первыми 10 предсказаниями

Кнопка для скачивания полных результатов в формате JSON

Скачайте результаты (при необходимости) или загрузите новый файл.
